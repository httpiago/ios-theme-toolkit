# Popover

To use this component, simply import it into any file and pass the [modifiers](#/#props) in the properties.

```jsx
import { Popover, Button } from 'ios-theme-toolkit'

const popoverContent = <span>Hello world</span>

export default () => (
  <Popover
    content={popoverContent}
  >
    <Button>Open popover</Button>
  </Popover>
)
```

#### Tip

If you put some element that can receive focus (buttons, inputs ...) inside the popover, place an "autofocus" attribute on the element to keyboard-friendly navigation.
