# List

To use this component, simply import it into any file and pass the [modifiers](#/#props) in the properties.

```jsx
import { List } from 'ios-theme-toolkit'

export default () => (
  <List>
    <List.Item>1</List.Item>
    <List.Item>2</List.Item>
    <List.Item>3</List.Item>
  </List>
)
```
