# Modal

To use this component, simply import it into any file and pass the [modifiers](#/#props) in the properties.

Note: This component has no state (a controlled component), you must provide the visibility state and close it "manually".

See more in the example below:

```jsx
import { Modal } from 'ios-theme-toolkit'

class ModalExample extends React.Component {
  state = {
    modalVisibility: false
  }

  openModal = () => {
    this.setState({ modalVisibility: true })
  }

  closeModal = () => {
    this.setState({ modalVisibility: false })
  }

  render() {
    return (
      <React.Fragment>
        <Modal
          visible={this.state.modalVisibility}
          onRequestClose={this.closeModal}
          title="Hello world!"
          content="OlÃ¡ mundo!"
          buttons={[
            <Button onClick={this.closeModal}>Close</Button>,
            <Button onClick={this.closeModal}>Hello</Button>,
          ]}
          style={{ width: 460 }}
        />

        <Button fill onClick={this.openModal}>Open modal</Button>
      </React.Fragment>
    )
  }
}
```
