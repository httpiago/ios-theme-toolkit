# Tooltip

To use this component, simply import it into any file and pass the [modifiers](#/#props) in the properties.

```jsx
import { Tooltip } from 'ios-theme-toolkit'

export default () => (
  <Tooltip text={'Hello world!'}>Hover me</Tooltip>
)
```
