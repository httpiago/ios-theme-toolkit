# Grouped buttons (segmented)

To use this component, simply import it into any file and pass the [modifiers](#/#props) in the properties.

```jsx
import { ButtonGroup, Button } from 'ios-theme-toolkit'

export default () => (
  <ButtonGroup>
    <Button>First</Button>
    <Button fill>Second</Button>
    <Button>Third</Button>
  </ButtonGroup>
)
```
