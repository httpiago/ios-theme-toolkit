# Input

To use this component, simply import it into any file and pass the [modifiers](#/#props) in the properties.

**NOTE: This package does not provide any form of data validation, I recommend you take a look at [formik](https://github.com/jaredpalmer/formik).**

```jsx
import { Input } from 'ios-theme-toolkit'

export default () => (
  <Input value="Hello world" />
)
```
